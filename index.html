Ошибки

Уходи, чудовище!

Вечером мир выглядит иначе. Таинственные тени появляются на стенах…Чудовища подкрадываются к кровати… Ты не можешь заснуть? Ты боишься монстров? Сыграй в игру и прoгони их в шкаф!

ХОД ИГРЫ

Игрок старается найти игрушки, которые испугают монстров. Каждое чудовище боится только одной игрушки – той, которая изображена на его карте. Игрок открывает любой жетон с игрушкой. В зависимости от того, какой жетон будет открыт, игрок должен выполнить одно из 3-х действий:

1. Если игрок откроет игрушку идентичную той, которая изображена на карте монстра(одного из монстров): • Испуганное чудовище идет спать. • Жетон с игрушкой вновь переворачивается.

2. Если игрушка не испугает монстров (т.е. не совпадет ни с одной изображенной игрушкой на карточке чудовищ): • Переворачивается одна из частей монстра в банке (начиная снизу). После трёх неудачных попыток найти нужную игрушку, когда появится целая фигурка монстра в банке, возле кровати должен появиться новый монстр.

3. Если открытый жетон является одним из двух жетонов анти-Джокеров: • Рисунки на жетонах анти-Джокерах изображены на цветном фоне. Когда время игры жетоны открываются впервые, ничего не происходит. Просто, нужно перевернуть жетон обратно рисунком вниз, запомнить его местоположение. Не нужно переворачивать ни одну из частей монстра в банке. Если игрок в очередной раз откроют один из специальных жетонов, нужно выполнить следующие действия:

Носки

Носки не испугает никакого монстра, а наоборот создаст беспорядок! Один из жетонов с игрушкой меняется местами с жетоном с носками. Затем, переворачивается очередную часть монстра в банке

Серенький волчок

Все части монстра в банке, нужные для появления нового монстра переворачиваются, новый монстр немедленно выходит.

КОНЕЦ ИГРЫ

Игрок побеждает в том случае, если ему удастся загнать всех монстров в шкаф! Когда все четыре места вокруг кровати заняты чудовищами, побеждают чудовища. Если монстры выиграли эту партию, лучший вариант – начать новую игру с целью взять реванш.
